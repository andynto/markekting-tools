# 金御賞互動式行銷內容產生器

這是純前端 PWA 版本，不需要 OpenAI API Key，不需要 Vercel，也不會產生 API 費用。

## 功能
- 品牌：金御賞集團、金子半之助、辻半、宇奈鰻魚飯
- 行銷主題：新品上市、季節限定、週年慶、會員活動、媒體邀約、顧客回訪等
- 產出：Facebook、Instagram、Threads、新聞稿、LINE OA、簡訊、短影音腳本、圖片提示詞、7 天排程
- 本機 Canvas 海報草稿，可下載 PNG
- 可安裝成 PWA，支援離線開啟

## GitHub Pages 使用方式
1. 解壓縮 ZIP。
2. 把資料夾內所有檔案上傳到 GitHub Repository 根目錄。
3. 到 Settings → Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main / root。
6. 儲存後等待 GitHub Pages 部署完成。

## 注意
此版本是規則式互動產生器，內容不會呼叫 ChatGPT API。優點是簡單、免費、可直接發佈；缺點是文案變化不如真正串接 AI API 的版本。
